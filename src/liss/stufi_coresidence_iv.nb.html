The design compares adjacent study-entry cohorts around the 2015 abolition of the Dutch basic study grant. The main idea is that cohorts just after the reform faced weaker financial support, which should raise the incentive to stay with parents while studying.

# Cohorts are defined by the year respondents turn 18 and start the September
# study year. We keep only respondents observed both at study start and again
# at age 30 so the instrument and outcome are defined on the same sample.
find_respondents_in_waves <- function(df, birth_year, waves) {
  df %>%
    filter(birth_year == .env$birth_year) %>%
    filter(wave %in% as.Date(waves)) %>%
    distinct(pid, wave) %>%
    count(pid) %>%
    filter(n == length(waves)) %>%
    pull(pid)
}

extract_model_term <- function(model, pattern) {
  term_name <- grep(pattern, names(coef(model)), value = TRUE)[1]

  if (is.na(term_name)) {
    return(tibble(term = NA_character_, estimate = NA_real_, std.error = NA_real_))
  }

  coef_table <- as.data.frame(coeftable(model), stringsAsFactors = FALSE)
  coef_table$term <- rownames(coef_table)

  coef_table %>%
    as_tibble() %>%
    filter(term == .env$term_name) %>%
    transmute(
      term,
      estimate = Estimate,
      std.error = `Std. Error`
    )
}

extract_wald_stat <- function(model, variable) {
  stat <- tryCatch(fixest::wald(model, variable), error = function(e) NULL)

  if (is.null(stat)) {
    return(tibble(statistic = NA_real_, p.value = NA_real_))
  }

  tibble(
    statistic = unname(stat["stat"]),
    p.value = unname(stat["p"])
  )
}

extract_rd_effect <- function(rd_object, label) {
  tibble(
    specification = label,
    estimate = unname(rd_object$coef[1]),
    std.error = unname(rd_object$se[1]),
    ci_low = estimate - 1.96 * std.error,
    ci_high = estimate + 1.96 * std.error
  )
}

run_placebo_iv <- function(data, cutoff_year) {
  placebo_data <- data %>%
    mutate(
      running_var_placebo = cohort_year - cutoff_year,
      post_placebo = as.integer(cohort_year >= cutoff_year)
    )

  feols(
    log_income_30 ~ wo + hbo + female + running_var_placebo |
      years_coresident ~ post_placebo,
    data = placebo_data,
    vcov = "HC1"
  )
}

run_subsample_iv <- function(data, label) {
  model <- feols(
    log_income_30 ~ wo + hbo + running_var |
      years_coresident ~ post_reform,
    data = data,
    vcov = "HC1"
  )

  extract_model_term(model, "years_coresident") %>%
    mutate(
      group = label,
      ci_low = estimate - 1.96 * std.error,
      ci_high = estimate + 1.96 * std.error
    )
}

  filter(age == 30) %>%

OLS benchmark

The OLS regression is only a descriptive benchmark. It likely mixes the causal effect of co-residence with selection into living at home.

First stage

The first stage tests whether the 2015 reform shifted the amount of time students lived with parents. If the instrument is weak, the IV estimate becomes unreliable.

Reduced form

The reduced form measures whether cohorts exposed to the reform earn differently at age 30, before scaling by the first-stage shift in co-residence.

IV / 2SLS

The IV estimate isolates the component of parental co-residence induced by the grant reform. Under the exclusion restriction, this gives the local causal effect for compliers around the cutoff.

Main results table

Table 2 puts the benchmark OLS, first stage, reduced form, and IV estimates side by side. All reported standard errors are HC1 robust.

RD estimation

The regression discontinuity design uses cohort year as the running variable and the 2015 policy change as the cutoff. The first stage, reduced form, and fuzzy RD versions mirror the linear specifications above but localize the comparison near the cutoff.

Figures

The figures visualize the discontinuity, the event-study pattern across cohorts, the density test, and sensitivity to the bandwidth choice.

Robustness checks

The placebo cutoffs check whether the estimated discontinuity appears where no reform happened. The donut specification drops cohorts immediately around the cutoff to see whether the main result is driven only by observations closest to the threshold.

Heterogeneity

Following your instruction to drop parental income, heterogeneity is shown only by sex. This checks whether the local effect differs for women and men.

Interpretation checklist

The causal interpretation is strongest if three pieces line up:

  1. The first stage is strong, so the reform meaningfully changed co-residence.
  2. Pre-reform cohorts look smooth in the event-study and placebo tests.
  3. The McCrary test does not suggest sorting around the cutoff.

If these checks fail, the IV estimate should be treated more cautiously.
